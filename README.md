# 项目简介

## 核心功能
- 个人中心
- 用户管理
- 物业管理
- 积分充值
- 垃圾站点管理
- 垃圾分类
- 垃圾知识库
- 垃圾识别
- 垃圾投放
- 垃圾验收
- 商品分类
- 积分商品
- 积分兑换
- 公告信息
- 留言板
- 系统管理

## 适用场景
- 社区环境下的垃圾管理
- 垃圾分类推广
- 环保意识提升

## 技术特点
- 架构：B/S、MVC、Web框架
- 开发环境：IDEA、JDK1.8、Mysql、Maven、Navcat
- 技术栈：Java、SpringBoot、Vue
